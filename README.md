# Biking Sales Excel Analysis Project (Feb. 2026)

:thought_balloon: **Context of Project:**
- Using dataset table of Bike Sales including customer infgormation such as:
  1. Age
  2. Home owners
  3. Income
  4. Education
  5. Marital Status
  6. Occupation
  7. Commute distane

:bookmark_tabs: **Process of Project**
- Cleaned dataset and organized customers by Age Brackets to better identify trends and discrepancies.
- Converted dataset into working a working table to filter for specific information when needed.
- Created Pivot Tables to analyze dataset, looking into customer sales based off comute distance, average income by gender, count of purhcases made by age bracket etc.
- Charted Pivot Tables to visualize trends within the data
- Created Excel Dashboard to summarize findings incorporating slicers for Education, Region, and Marital Status.
